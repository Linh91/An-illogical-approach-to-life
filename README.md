# An-illogical-approach-to-life

A turn based rpg game 

## Tech Stack
AngularJS
Express
Node.js
MongoDB
HTML
CSS

## Run

Install node

Install npm dependencies:

    $ cd path/to/project/root
    $ npm install

Run dev server:

    $ cd path/to/project/root
    $ node server
